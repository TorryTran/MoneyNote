# MoneyNote

MoneyNote là ứng dụng Android quản lý thu chi cá nhân, tập trung vào tốc độ nhập liệu, theo dõi trực quan và trải nghiệm sử dụng đơn giản mỗi ngày.

Ứng dụng lưu dữ liệu offline trên thiết bị, hỗ trợ quản lý ví tiền, xem lịch sử giao dịch theo lịch, theo dõi tổng quan thu chi, ghi chú cá nhân và sao lưu/khôi phục dữ liệu khi cần.
